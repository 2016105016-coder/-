# Элиас: обществознание 10

Нативное Android-приложение для командной игры по обществознанию для 10 класса.

## Что уже есть

- цветное стартовое меню;
- 2–6 команд с редактируемыми названиями;
- три уровня сложности: лёгкий, средний, сложный;
- игровой экран с таймером 60 секунд;
- подсчёт очков по командам;
- победа при достижении 20 очков;
- звуки: клик, правильный ответ, пропуск, завершение времени;
- встроенный словарь карточек по обществознанию;
- правила игры внутри приложения.

## Как открыть

1. Распакуйте архив.
2. Откройте папку `alias_obshestvoznanie_10` в Android Studio.
3. Дождитесь синхронизации Gradle.
4. Запустите приложение на эмуляторе или Android-телефоне.

## Технически

- Язык: Java.
- UI: программный Android View API без внешних библиотек.
- Минимальная версия Android: API 23.
- Compile SDK: 36.
- Android Gradle Plugin: 8.13.2.

Если Android Studio предложит установить нужный SDK или обновить Gradle, можно согласиться. Если в установленной Android Studio отсутствует SDK 36, в `app/build.gradle` можно временно заменить `compileSdk 36` и `targetSdk 36` на установленную версию, например 35.

## Где менять карточки

Список терминов расположен в файле:

`app/src/main/java/ru/school/aliassociety/MainActivity.java`

Нужный блок — метод `initCards()`. Карточка добавляется строкой:

```java
add(MEDIUM, "термин", "подсказка для ведущего");
```

Уровни:

```java
EASY
MEDIUM
HARD
```
