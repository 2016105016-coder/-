# 3D модель: город будущего

Содержимое:

- `future_city.obj` — 3D-модель города будущего;
- `future_city.mtl` — материалы (цвета) для модели;
- `generate_future_city_obj.py` — скрипт генерации модели;
- `preview.html` — визуальный WebGL-просмотр модели в браузере.

## Что есть в сцене

- небоскребы разной высоты;
- зелёные парки с деревьями;
- воздушные трассы;
- летающие машины;
- посадочные площадки на крышах высоких зданий.

## Где посмотреть визуально

### Вариант 1: быстро в браузере (рекомендуется)

Из папки `assets/future_city` запусти локальный сервер:

```bash
python3 -m http.server 8080
```

Открой в браузере:

```text
http://localhost:8080/preview.html
```

### Вариант 2: в Blender

1. `File` → `Import` → `Wavefront (.obj)`.
2. Выбери `future_city.obj` (материалы из `future_city.mtl` подтянутся автоматически).

## Как пересобрать модель

```bash
python3 generate_future_city_obj.py
```

После запуска скрипт перезапишет `future_city.obj` и `future_city.mtl`.
